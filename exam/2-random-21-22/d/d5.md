## 题目
已知二维离散型随机变量 $(X,Y)$ 的联合概率分布如表：

| $X$ 取值 | $Y=-1$ | $Y=0$ | $Y=1$ |
|----------|--------|-------|-------|
| 2        | 0.2    | 0.1   | 0     |
| 4        | 0      | $a$   | 0.1   |
| 6        | 0.1    | 0     | 0.2   |

求：（1）常数 $a$；（2）$X,Y$ 的边缘分布律；（3）判断 $X$ 与 $Y$ 是否相互独立。

## 步骤

**（1）求常数 $a$**  
联合概率分布的所有概率之和为 1：

$\displaystyle\sum_{i}\sum_{j} P(X=x_i, Y=y_j) = 1$

**（2）求边缘分布律**  
$X$ 的边缘分布：$P(X=x_i) = \displaystyle\sum_{j} P(X=x_i, Y=y_j)$  
$Y$ 的边缘分布：$P(Y=y_j) = \displaystyle\sum_{i} P(X=x_i, Y=y_j)$

**（3）独立性判定**  
$X$ 与 $Y$ 相互独立当且仅当对所有 $i,j$ 都有：

$P(X=x_i, Y=y_j) = P(X=x_i) \cdot P(Y=y_j)$

只要存在一对 $(i,j)$ 使等式不成立，即可判定不独立。

## 题解

**（1）常数 $a$**

$0.2 + 0.1 + 0 + 0 + a + 0.1 + 0.1 + 0 + 0.2 = 1$

$0.7 + a = 1$

$a = 0.3$

**（2）边缘分布律**

$X$ 的边缘分布：

$P(X=2) = 0.2 + 0.1 + 0 = 0.3$  
$P(X=4) = 0 + 0.3 + 0.1 = 0.4$  
$P(X=6) = 0.1 + 0 + 0.2 = 0.3$

$Y$ 的边缘分布：

$P(Y=-1) = 0.2 + 0 + 0.1 = 0.3$  
$P(Y=0) = 0.1 + 0.3 + 0 = 0.4$  
$P(Y=1) = 0 + 0.1 + 0.2 = 0.3$

**（3）独立性判断**

取 $X=2,\ Y=0$ 检验：

$P(X=2, Y=0) = 0.1$  
$P(X=2) \cdot P(Y=0) = 0.3 \times 0.4 = 0.12$

$0.1 \neq 0.12$，因此 $X$ 与 $Y$ **不相互独立**。
