# 原子卡制作规范

## 格式

每条卡片为三行，依次是 `主题`、`求解`、`路径`，卡间空一行：

```
主题：<卡片标题>
求解：<待解决的问题>
路径：<解法/公式/要点>
```

## 原子性

- **够小、够细、够原子**：一张卡片只讲一个概念或一个步骤
- 不在一张卡里塞多个知识点
- 如果一步可以拆成两步，就拆成两张卡

## 制卡过程
先生成相应的例题和解答，然后再检查自己生成的内容，提取出流程线，分解出原子步骤，并对每个步骤都制作一个卡片，使得卡片顺序就是步骤顺序。

## 示例

```
主题：切比雪夫
求解：开始
路径：算期望和方差
```

```
主题：切比雪夫
求解：算期望和方差后
路径：X-E(X)小于某个范围，这个概率大于 1 - 方差除以范围的平方
```

```
主题：标准误
求解：SE
路径：σ/√n（Z）或 s/√n（t）
```

## 递进关系

- 多张卡按解题/推理顺序排列
- 前一张的输出是后一张的输入
- 整体构成一条完整的问题解决路径
