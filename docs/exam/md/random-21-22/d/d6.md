## 理解
8xy 是权重，而 x 是连续化的 x 类下的对象，对象乘以权重来看最终的概率偏向于哪个对象，即加权平均。
E(XY)−E(X)E(Y) 中，E(XY) 是所有点真实值的平均，E(X)E(Y) 是所有点期望值的平均，差值直接就是协方差。而 E[(X−E(X))(Y−E(Y))] 则是先分别对每个点取差值，而这里期望不是用在具体的点上的，而是用在所有差值上的。

## 题目
设随机变量 $(X,Y)$ 的联合概率密度函数为

$$
f(x,y)=
\begin{cases}
8xy, & 0\le x\le 1,\ 0\le y\le x,\\
0, & \text{其他},
\end{cases}
$$

求：（1）$E(X)$，$E(Y)$；（2）$\operatorname{Cov}(X,Y)$。

## 步骤
**（1）求期望**  
边缘期望通过联合密度直接计算：

$E(X) = \displaystyle\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} x\,f(x,y)\,\mathrm{d}y\mathrm{d}x$

$E(Y) = \displaystyle\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} y\,f(x,y)\,\mathrm{d}y\mathrm{d}x$

积分区域为 $0\le x\le 1$，$0\le y\le x$。

**（2）求协方差**  
$\operatorname{Cov}(X,Y) = E(XY) - E(X)E(Y)$

其中 $E(XY) = \displaystyle\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} xy\,f(x,y)\,\mathrm{d}y\mathrm{d}x$。

## 题解
**（1）期望**

$E(X)$：

$\displaystyle E(X) = \int_{0}^{1}\int_{0}^{x} x\cdot 8xy\,\mathrm{d}y\mathrm{d}x
= \int_{0}^{1}\int_{0}^{x} 8x^{2}y\,\mathrm{d}y\mathrm{d}x$

$\displaystyle = \int_{0}^{1} \left[4x^{2}y^{2}\right]_{y=0}^{y=x} \mathrm{d}x
= \int_{0}^{1} 4x^{4}\,\mathrm{d}x
= 4\left[\frac{x^{5}}{5}\right]_{0}^{1}
= \frac{4}{5}$

$E(Y)$：

$\displaystyle E(Y) = \int_{0}^{1}\int_{0}^{x} y\cdot 8xy\,\mathrm{d}y\mathrm{d}x
= \int_{0}^{1}\int_{0}^{x} 8xy^{2}\,\mathrm{d}y\mathrm{d}x$

$\displaystyle = \int_{0}^{1} \left[\frac{8x}{3}y^{3}\right]_{y=0}^{y=x} \mathrm{d}x
= \int_{0}^{1} \frac{8x^{4}}{3}\,\mathrm{d}x
= \frac{8}{3}\left[\frac{x^{5}}{5}\right]_{0}^{1}
= \frac{8}{15}$

所以 $E(X)=\dfrac{4}{5}$，$E(Y)=\dfrac{8}{15}$。

**（2）协方差**

先求 $E(XY)$：

$\displaystyle E(XY) = \int_{0}^{1}\int_{0}^{x} xy\cdot 8xy\,\mathrm{d}y\mathrm{d}x
= \int_{0}^{1}\int_{0}^{x} 8x^{2}y^{2}\,\mathrm{d}y\mathrm{d}x$

$\displaystyle = \int_{0}^{1} \left[\frac{8x^{2}}{3}y^{3}\right]_{y=0}^{y=x} \mathrm{d}x
= \int_{0}^{1} \frac{8x^{5}}{3}\,\mathrm{d}x
= \frac{8}{3}\left[\frac{x^{6}}{6}\right]_{0}^{1}
= \frac{8}{18}
= \frac{4}{9}$

代入协方差公式：

$\displaystyle \operatorname{Cov}(X,Y) = E(XY) - E(X)E(Y)
= \frac{4}{9} - \frac{4}{5}\times\frac{8}{15}
= \frac{4}{9} - \frac{32}{75}
= \frac{100}{225} - \frac{96}{225}
= \frac{4}{225}$

因此 $\operatorname{Cov}(X,Y)=$ **$\dfrac{4}{225}$**。
