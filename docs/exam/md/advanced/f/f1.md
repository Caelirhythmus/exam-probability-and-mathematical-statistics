## 题目
设 $X_1,X_2,\ldots,X_n$ 为来自总体 $N(\mu,\sigma^2)$ 的简单随机样本，其中 $\sigma^2$ 已知。记样本均值为 $\overline{X}$，并考虑 $\mu$ 的另一个线性估计量

$$
\hat{\mu} = \frac{2}{n(n+1)}\sum_{i=1}^{n} i X_i.
$$

(1) 证明：$\overline{X}$ 与 $\hat{\mu}$ 均为 $\mu$ 的无偏估计；
(2) 计算 $D(\overline{X})$ 与 $D(\hat{\mu})$，并比较二者大小，说明哪个估计量更有效。

## 步骤
**（1）无偏性**  
验证 $E(\overline{X})=\mu$ 和 $E(\hat{\mu})=\mu$。利用 $E(X_i)=\mu$ 及求和公式 $\sum_{i=1}^{n} i = \dfrac{n(n+1)}{2}$。

**（2）有效性**  
计算 $D(\overline{X})$ 和 $D(\hat{\mu})$。利用 $D(X_i)=\sigma^2$ 及 $\sum_{i=1}^{n} i^2 = \dfrac{n(n+1)(2n+1)}{6}$。比较方差大小，方差越小越有效。

## 题解
**（1）无偏性**

对 $\overline{X}$：

$\displaystyle E(\overline{X}) = E\!\left(\frac{1}{n}\sum_{i=1}^{n} X_i\right)
= \frac{1}{n}\sum_{i=1}^{n} E(X_i)
= \frac{1}{n}\cdot n\mu = \mu$

故 $\overline{X}$ 是 $\mu$ 的无偏估计。

对 $\hat{\mu}$：

$\displaystyle E(\hat{\mu}) = E\!\left(\frac{2}{n(n+1)}\sum_{i=1}^{n} i X_i\right)
= \frac{2}{n(n+1)}\sum_{i=1}^{n} i\,E(X_i)$

$\displaystyle = \frac{2}{n(n+1)}\cdot\mu\sum_{i=1}^{n} i
= \frac{2}{n(n+1)}\cdot\mu\cdot\frac{n(n+1)}{2}
= \mu$

故 $\hat{\mu}$ 也是 $\mu$ 的无偏估计。

**（2）方差比较**

$\overline{X}$ 的方差：

$\displaystyle D(\overline{X}) = D\!\left(\frac{1}{n}\sum_{i=1}^{n} X_i\right)
= \frac{1}{n^2}\sum_{i=1}^{n} D(X_i)
= \frac{1}{n^2}\cdot n\sigma^2
= \frac{\sigma^2}{n}$

$\hat{\mu}$ 的方差：

$\displaystyle D(\hat{\mu}) = D\!\left(\frac{2}{n(n+1)}\sum_{i=1}^{n} i X_i\right)
= \frac{4}{n^2(n+1)^2}\sum_{i=1}^{n} i^2\,D(X_i)$

$\displaystyle = \frac{4\sigma^2}{n^2(n+1)^2}\cdot\frac{n(n+1)(2n+1)}{6}
= \frac{2\sigma^2(2n+1)}{3n(n+1)}$

比较二者比值：

$\displaystyle \frac{D(\hat{\mu})}{D(\overline{X})}
= \frac{2\sigma^2(2n+1)}{3n(n+1)}\Big/\frac{\sigma^2}{n}
= \frac{2(2n+1)}{3(n+1)}$

对任意 $n\ge 2$，有 $\dfrac{2(2n+1)}{3(n+1)} > 1$（$n=1$ 时等于 $1$），即 $D(\hat{\mu}) > D(\overline{X})$。

因此 $\overline{X}$ 的方差更小，**$\overline{X}$ 是比 $\hat{\mu}$ 更有效的估计量**。
