## 题目
已知某种零件的尺寸服从正态分布 $N(\mu,\sigma^2)$，其中 $\mu,\sigma^2$ 均未知。现随机抽取16件，测得样本均值 $\bar{x}=12.5\,\text{cm}$，样本标准差 $s=0.8\,\text{cm}$。取显著性水平 $\alpha=0.05$：
(1) 求 $\mu$ 的置信水平为 $1-\alpha$ 的置信区间；
(2) 检验假设 $H_0:\sigma^2=0.5$ vs $H_1:\sigma^2\neq 0.5$。

## 步骤
**（1）$\mu$ 的置信区间**  
$\sigma^2$ 未知，使用 $t$ 分布构造置信区间：

$\displaystyle \left(\overline{X} - t_{\alpha/2}(n-1)\,\frac{S}{\sqrt{n}},\ \overline{X} + t_{\alpha/2}(n-1)\,\frac{S}{\sqrt{n}}\right)$

**（2）$\sigma^2$ 的双侧检验**  
检验统计量：$\chi^2 = \dfrac{(n-1)S^2}{\sigma_0^2}\sim\chi^2(n-1)$

拒绝域：$\chi^2 < \chi^2_{1-\alpha/2}(n-1)$ 或 $\chi^2 > \chi^2_{\alpha/2}(n-1)$

## 题解
已知 $n=16$，$\overline{x}=12.5$，$s=0.8$，$\alpha=0.05$。

**（1）$\mu$ 的置信区间**

查 $t$ 分布表得 $t_{0.025}(15)=2.1314$。

标准误：$\displaystyle\frac{s}{\sqrt{n}} = \frac{0.8}{\sqrt{16}} = 0.2$

边际误差：$t_{0.025}(15)\times\dfrac{s}{\sqrt{n}} = 2.1314\times 0.2 = 0.4263$

置信区间：

$(12.5 - 0.4263,\ 12.5 + 0.4263) = (12.0737,\ 12.9263)$

因此 $\mu$ 的 $95\%$ 置信区间约为 **$(12.07,\ 12.93)$**。

**（2）$\sigma^2$ 的假设检验**

检验统计量的观测值：

$\displaystyle\chi^2 = \frac{(n-1)s^2}{\sigma_0^2}
= \frac{15 \times 0.8^2}{0.5}
= \frac{15 \times 0.64}{0.5}
= 19.2$

查 $\chi^2$ 分布表得临界值：

$\chi^2_{0.025}(15) = 6.2621,\qquad \chi^2_{0.975}(15) = 27.4884$

由于 $6.2621 < 19.2 < 27.4884$，检验统计量落在接受域内，故在显著性水平 $\alpha=0.05$ 下**不拒绝** $H_0$，即没有充分证据认为方差 $\sigma^2$ 显著不等于 $0.5$。
