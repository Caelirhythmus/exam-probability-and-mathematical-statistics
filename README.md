## 目录结构

## 命名规则

文件前缀表示试卷上的大题号：
- `a` — 第一大题（填空题）
- `d` — 第四大题（解答题）
- `e` — 第五大题（解答题）

后缀数字表示大题内的小题号。例如 `a3.md` 表示第一大题第 3 小题。

## 体例

每题按「题目 → 步骤 → 题解」三段式组织。
